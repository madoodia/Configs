====================
Building and Running And Debugging PBRT-V3 (https://github.com/mmp/pbrt-v3/)

in Visual Studio (x64)
$cd pbrt-v3
$mkdir build
$cd build
$call "%VCVARS_LOCATION%/vcvarsall.bat" x64
$cmake -G "Visual Studio 16 2019" -DCMAKE_BUILD_TYPE=Release ..

- Open PBRT-V3.sln
- Build ALL_BUILD in Debug Mode
- Set pbrt_exe project as Default project
- Set Debugging Arguments in pbrt_exe project's properties as: --nthread 1 D:\path\to\scenes\scene.pbrt
- Set breakpoint on SamplerIntegrator::Render()
- Run the project
